Algoritmo "CadastroBiblioteca"

Início
    // Entrada de dados
    Escreva("Digite o título do livro: ")
    Leia(tituloLivro)

    Escreva("Digite o autor: ")
    Leia(autor)

    Escreva("Digite o ano de publicação: ")
    Leia(anoPublicacao)

    Escreva("O livro está disponível? (verdadeiro/falso): ")
    Leia(disponivel)

    // Saída organizada
    Escreva("Cadastro realizado com sucesso!")
    Escreva("Título: ", tituloLivro)
    Escreva("Autor: ", autor)
    Escreva("Ano: ", anoPublicacao)
    Escreva("Disponível: ", disponivel)
Fim


## Pontos importantes

- **Variáveis:** `tituloLivro`, `autor`, `anoPublicacao`, `disponivel`.

- **Tipos de dados:**
  - `caractere` → texto
  - `inteiro` → números inteiros
  - `lógico` → verdadeiro/falso

- **Entrada:** o usuário fornece os dados.  

- **Processamento:** armazenamos cada informação em uma variável.  

- **Saída:** exibimos uma frase organizada com os dados cadastrados.
